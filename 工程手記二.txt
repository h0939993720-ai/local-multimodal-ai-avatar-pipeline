第五階段：FFmpeg 後處理問題
問題歷程
嘗試結果zoompan 全畫面晃動背景跟著動，穿幫rotate + crop 全畫面位移鏡頭晃動感，身體反而更僵stream_loop + vf 一段式FFmpeg 4.2 encoder EOF 錯誤兩段式展開再套濾鏡還是失敗純色彩後處理穩定，不移動畫面
結論
全畫面位移濾鏡會讓背景漂移，身體動作應完全交給 LivePortrait stitching 負責。FFmpeg 只做色彩微調：
python"hue=s=1.03,eq=contrast=1.02:brightness=0.01"

第六階段：FFmpeg 版本衝突問題
問題

venv312 內的 FFmpeg 是 4.2.2（舊版）
系統安裝的 FFmpeg 是 8.1（C:\ffmpeg\ffmpeg-8.1-essentials_build\bin\）
venv 環境優先使用舊版，觸發 not open encoder before EOF

解法：建立獨立 FFmpeg Server
ffmpeg_server.py（Port 7862）

系統 Python 執行，不需要任何 venv
直接呼叫 FFmpeg 8.1
提供三個路由：

POST /process_video：驅動影片裁切
POST /merge_audio：音訊合併
POST /extract_frame：截取單幀


prepare_driving_video.py 和 extract_source_frame.py 改為 HTTP 呼叫此 Server
